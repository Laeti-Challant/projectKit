📋 À propos

Cette application vous aide à créer un cahier des charges structuré pour vos projets web, étape par étape.

✨ Fonctionnalités

  • 📝 Formulaire guidé en 9 sections

  • 📄 Export PDF professionnel
  • 🔒 Vos données restent locales (localStorage)

🚀 Technologies

Projet créé avec JavaScript vanilla, HTML5 et CSS3

📖 Comment utiliser ?

  1. Remplissez les sections une par une
  2. Vos données sont sauvegardées automatiquement
  3. Exportez votre document en PDF quand vous avez terminé
